# PocketMoney
심부름, 알바 등 다양한 구인글을 올려서 용돈벌이를 할 수 있는 사이트

## 💻 기술 스택 & 개발 환경
### 프레임워크/API
- Backend  : Spring Boot, Spring Data JPA, Spring Security, Kakao Login API  
- Frontend : React 
### 데이터베이스
- MariaDB(실서버), H2DB(테스트용)
### 언어
- Backend : Java
- Frontend : HTML/CSS/JS
- Database : SQL
